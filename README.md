# Video-of-Ascii
python版 视频转Ascii流

## 分步
- 第一步：```video_to_jpg.py```是可以把视频转化为一帧一帧的jpg
- 第二步：```jpg_to_ascii.py```是可以把图片转化成ascii码组成的txt
- 第三步：将全部的txt在终端快速打印出来，即可展示为ascii流视频

## 单步
- 打开terminal
```shell script
python main.py video.mp4
```


## 利用```video_to_ascii```包
```shell script
pip install video_to_ascii
  
video-to-ascii -f myvideo.mp4
```
